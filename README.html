Monitoramento de Câmeras com Análise de Placas

Descrição do Projeto

Este projeto é uma aplicação de monitoramento de câmeras que utiliza o YOLOv8 para detectar objetos e o EasyOCR para reconhecer placas de veículos. Ele também faz a integração com um banco de dados PostgreSQL para registrar entradas e saídas de veículos com base na leitura das placas.

Requisitos

  • Python 3.8 ou superior
  • OpenCV
  • pandas
  • ultralytics (YOLOv8)
  • numpy
  • easyocr
  • Pillow (PIL)
  • psycopg2
  • tkinter (já incluído no Python padrão)
  • PostgreSQL

Instalação

  1. Clone o repositório:
    2. git clone https://github.com/seuusuario/seurepositorio.git
cd seurepositorio
  2. Crie um ambiente virtual e ative-o:
    3. python -m venv venv
source venv/bin/activate  # Para Windows, use `venv\Scripts\activate`
  3. Instale as dependências:
    4. pip install opencv-python pandas ultralytics numpy easyocr Pillow psycopg2
  4. Configure o PostgreSQL:
    • Certifique-se de que o PostgreSQL está instalado e em execução.
    • Crie um banco de dados chamado controle.
    • Ajuste o nome do banco de dados, usuário e senha no script, se necessário.

Uso

  1. Prepare o Modelo e Arquivos de Configuração:
    • Coloque o modelo YOLOv8 pré-treinado em modelo/best.pt.
    • Crie um arquivo de texto coco1.txt com a lista de classes que o modelo pode detectar.
  2. Crie as Tabelas no Banco de Dados:
    • O script camera_app.py criará automaticamente as tabelas necessárias quando for executado.
  3. Execute o Aplicativo:
    • Certifique-se de que o arquivo de vídeo resources/mycarplate.mp4 esteja no diretório correto.
    • Execute o script principal:
      • python camera_app.py
    • O aplicativo abrirá uma janela com o monitoramento das câmeras.
  4. Interação com o Relatório:
    • Para abrir a janela de relatórios, certifique-se de que o módulo relatorio está disponível e que a função open_report_window() está implementada no arquivo relatorio.py.

Estrutura do Projeto

project/
│
├── camera_app.py      # Script principal da aplicação
├── modelo/
│   └── best.pt        # Modelo pré-treinado YOLOv8
├── resources/
│   └── mycarplate.mp4 # Vídeo para teste
├── coco1.txt          # Lista de classes para YOLO
└── relatorio.py       # Módulo para geração de relatórios (opcional)

Funções Principais

  • Leitura e Processamento de Vídeo: Captura vídeo de fontes especificadas e processa cada frame.
  • Detecção de Objetos: Usa YOLOv8 para detectar objetos nas imagens.
  • Reconhecimento de Placas: Usa EasyOCR para ler placas de veículos.
  • Registro em Banco de Dados: Insere e atualiza registros de entrada e saída no banco de dados PostgreSQL.
  • Interface Gráfica: Interface com o usuário utilizando tkinter.
